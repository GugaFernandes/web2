lsita de comandos do git

  • git init - cria um repositorio em uma pasta vazia
  • git add - adiciona modificações do wk no stage
  • git status - mostra o estado atual dos repositórios, modificações, arquivos versionados
  • git commit - cria um commit no repositorio LOCAL
  • git push - envia modificações do repositorio LOCAL para o remoto
  • git log - exibe no terminal os commits do repositorio na branch atual
  • git log --online --all - exibe todos os commits de forma resumida
  • git branch - lista de branches no repositorio
  • git checkout - muda de branch
  • git checkout -b - cria e muda para uma nova branch
